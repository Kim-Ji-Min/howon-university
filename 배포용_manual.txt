2. 이미지 수정/교체 

1-1. 홈페이지 접속 - 교체할 사진 탐색 -  오른쪽 클릭 후 저장 (※반드시 이름 그대로 저장)
1-2. 홈페이지 접속 - 교체할 사진 탐색 - 웹FTP img 폴더에서 찾아 다운로드 (※반드시 이름 그대로 저장)
2. 이미지 교체 작업 ( 포토샵을 이용을 추천, ctrl + alt + shift + s [save for web] 이미지속성 gif )
3. 관리홈 - 왼쪽메뉴 하단 디자인관리 - 웹디자인FTP - PC웹 디자인 FTP접속 - img폴더 클릭 - select files - 
교체파일 클릭 - upload files - 완료

※ 서버지연시 바로 교체 되지 않을 수 있습니다. (1~5분정도 소요)
다수의 시간이 지나도 교체되지 않을 시, 원본 파일 삭제 후 같은 방법으로 재 업로드 바랍니다.
이름, 확장자명이 불일치 할 경우 오류발생하므로 주의
원본파일)sub_main.gif → 수정파일 저장) sub_main.gif(o) sub_main.jpg(x) sub-main.gif(x)

※ 웹의 접근성을 위하여 저장시 72dpi, gif 또는 jpg 파일을 권장합니다.
